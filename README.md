**weather_agent**

Weather Agent using Python & Open-Meteo API

This project is a simple weather assistant that fetches real-time temperature and windspeed for major Indian cities using the Open-Meteo API.

**Features**:

Fetch live weather data

Temperature & windspeed retrieval

API integration using requests

Dictionary-based city lookup

Beginner-friendly code

**How the project works**:

The user enters a city name

The city is mapped to latitude & longitude

Open-Meteo API is called

JSON response is parsed

Final weather report is returned
